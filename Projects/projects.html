Motion Keyboard for Android

A sensor controlled keyboard for Android devices targeted at users with physical disabilities. Its main functionality allows phone's built-in motion sensor to guide a cursor on top of the phone's keyboard, enabling the user to use hand motions to select various keys on the keyboard. Additional functionalities include word predictions, different keyboard layouts and more.
